## Shipment Tracking at Scale

At Cubbo, I designed and delivered an end-to-end shipment tracking page that serves around 500K+ users annually. 

The project involved integrating with various external APIs to fetch real-time tracking data and presenting it in a user-friendly interface built with React. The backend was developed using Ruby on Rails, with PostgreSQL as the database to manage shipment information efficiently. The system was deployed on AWS, ensuring scalability and reliability to handle high traffic volumes. 

This project significantly improved the customer experience by providing accurate and timely tracking information, enhancing transparency and trust to Cubbo's logistics services.
