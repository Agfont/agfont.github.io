## Healthy Market E-commerce Store

Healthy Market is a custom headless Shopify e-commerce store specializing in organic food products. 

The project involved designing and developing a user-friendly online shopping experience that seamlessly integrates with Shopify's API. The frontend was built using Next.js and Tailwind CSS, providing a responsive and visually appealing interface. GraphQL was utilized for efficient data fetching, while React and Node.js were used to create dynamic features and ensure smooth performance. 

The result is a modern, fast, and engaging e-commerce platform that caters to health-conscious consumers looking for organic products.
